# Smart Door Lock with Fingerprint Sensor

## Hardware Required
- Arduino Uno/Nano
- R307 Fingerprint Sensor
- 12V Solenoid Lock
- Relay Module
- 5V Power Supply
- 12V Power Supply
- LEDs and Resistors
- Jumper Wires

## Wiring Diagram
| Component | Arduino Pin |
|-----------|-------------|
| Fingerprint RX | Pin 2 |
| Fingerprint TX | Pin 3 |

2. Send '1' to enter enrollment mode
3. Follow prompts to add fingerprint
